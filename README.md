# PlantCare

Веб-приложение для отслеживания расписания ухода за комнатными растениями. Проект разработан для автоматизации напоминаний о поливе, визуализации статистики коллекции и удобного управления данными через локальное хранилище браузера.

## О проекте

PlantCare позволяет пользователям:
*   Добавлять растения с указанием типа, даты посадки и частоты полива.
*   Отслеживать статус здоровья растений с помощью визуальных прогресс-баров.
*   Фильтровать коллекцию по состоянию ("Требуют полива", "Здоровы") или искать по названию.
*   Просматривать статистику коллекции (количество типов, потребность в воде).
*   Использовать адаптивный интерфейс на мобильных устройствах и планшетах.

##№ Стек технологий

*   **HTML5 / CSS3** (Custom Properties, Flexbox, Grid)
*   **JavaScript (ES6+)** (Modules, LocalStorage API)
*   **Jest** (Модульное тестирование)
*   **Babel** (Транспиляция кода для тестовой среды)

## Установка и запуск

1.  Клонируйте репозиторий или скачайте архив с проектом.
2.  Установите зависимости для запуска тестов:
    ```bash
    npm install
    ```
3.  Откройте файл `index.html` в любом современном браузере.


## Тестирование

Проект покрыт модульными тестами, проверяющими ключевую бизнес-логику и алгоритмы.

### Как запустить тесты:

```bash
npm test
```

### Особенности реализации 

- Адаптивная верстка: Интерфейс автоматически подстраивается под экраны телефонов, планшетов и декстопа

- Темная и светлая темы: Поддержка переключения между светлой и темной темой с сохранением выбора пользователя

- Пагинация: Удобное пролистывание карточек при большом количестве растений в коллекции

- Empty State: Подсказки, когда коллекция пуста или поиск не дал результатов
